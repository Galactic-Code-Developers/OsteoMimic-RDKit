# Contributing Guidelines

Contributions are welcome provided they align with the scientific scope
of this repository.

## Scope

This repository is limited to molecular-scale simulations and descriptor
analysis using RDKit. Contributions implying mechanical, biological, or
clinical performance should be supported by appropriate external validation
and are outside the intended scope.

## Workflow

1. Fork the repository
2. Create a feature branch
3. Commit changes with clear messages
4. Submit a pull request describing the modification

By contributing, you agree that your contributions will be released under
the repository license.
